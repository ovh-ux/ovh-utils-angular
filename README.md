## Installation

### NPM

```sh
$ npm install @ovh-ux/ng-ovh-utils --save # or yarn add @ovh-ux/ng-ovh-utils
```

## Get the sources

```sh
$ git clone https://github.com/ovh-ux/ng-ovh-utils.git
$ cd ng-ovh-utils
$ npm install # or yarn
```
